   - [ ] Given 第十局游戏， When STRIKE， Then 画面出现X且分数为10加上后两个球击倒的瓶数和
   
   - [ ] Given 第十局游戏， When SPARE， Then 画面出现\且分数为10加上后一个球击倒的瓶数和
